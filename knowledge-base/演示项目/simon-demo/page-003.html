# ChatLab 详细提取

## 安装方式

**桌面应用：**
从官网 (chatlab.fun) 或 GitHub Releases 下载安装包，双击安装

**命令行工具 (CLI)：**
- 要求：Node.js ≥ 20
- 安装：`npm i chatlab-cli -g`
- 启动命令：
  - `chatlab start` — 启动 API + Web UI，自动打开浏览器
  - `chatlab start --no-open` — 启动但不自动打开浏览器
  - `chatlab start --headless` — 仅 API，无 Web UI
  - `chatlab start --daemon` — 安装为系统服务（开机自启动）

## 支持的聊天平台

**当前支持：**
WhatsApp, LINE, QQ, Discord, Instagram, Telegram, iMessage, Google Chat

**即将支持：**
Messenger, KakaoTalk

## AI 分析功能

核心能力是 "Agent + Function Calling workflows (24+ tools) can search, summarize, and analyze chat records"

具体包括：
- 在聊天数据上进行搜索、总结和分析
- 探索对话模式
- 提出更好的问题并提取洞察
- 具备上下文感知能力

## 核心特性

- **大规模历史记录处理**：流式解析和多工作进程，即使百万级消息也能保持响应
- **默认隐私保护**：数据和设置保存在本地，无需上传原始对话到云端
- **数据可视化洞察**：趋势、时间模式、互动频率、排名等
- **跨平台标准化**：将不同导出格式映射到统一模型

## 核心工作流程

数据流动分为五个阶段：
**"format detection → stream parsing → local persistence → SQL + AI query → visualization"**

## 数据导入

页面提到有 "Chat Record Export Guide" 指南，说明需要先从各平台导出聊天记录，然后通过 ChatLab 导入。采用流式解析方式处理大文件。

**定位**：本地化的智能聊天记录分析平台（桌面应用 + Web UI）

## 📦 ChatLab 的使用方式

### 安装

**方式 1：桌面应用（推荐）**
```
1. 访问 chatlab.fun 或 GitHub Releases
2. 下载安装包（Windows/Mac/Linux）
3. 双击安装，启动即用
```

**方式 2：命令行（开发者）**
```bash
# 安装（需要 Node.js ≥ 20）
npm i chatlab-cli -g

# 启动（自动打开浏览器）
chatlab start

# 后台运行
chatlab start --daemon
```

### 使用流程

```mermaid
graph LR
    A[导出聊天记录] --> B[导入到 ChatLab]
    B --> C[流式解析]
    C --> D[本地持久化]
    D --> E1[SQL 查询]
    D --> E2[AI 智能分析]
    E1 --> F[可视化展示]
    E2 --> F
```

**5 步工作流程**：
1. **导出** - 从 WhatsApp/QQ/Discord 等平台导出聊天记录
2. **导入** - 拖拽文件到 ChatLab
3. **解析** - 流式处理（支持百万级消息）
4. **分析** - SQL 查询 + AI Agent（24+ 工具）
5. **可视化** - 趋势图、互动频率、排名等
