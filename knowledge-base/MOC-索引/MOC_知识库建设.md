## 业界对标
- **Quartz**: 开源静态知识库 SSG,原生支持双向链接/反链/图谱/搜索
- **Obsidian**: PKM 桌面应用,社区共识:文件夹 ≤2 层 + MOC + 标签
- **Dendron**: 点层次命名,适合 10,000+ 笔记,但需 VS Code 扩展支持
- **PARA 方法**: Projects / Areas / Resources / Archives,防止活跃文件膨胀

## 关键决策记录
1. **不迁移到 Quartz**(2026-07-01):保留"零构建 + 本地全功能 + 多格式预览"的独特优势
2. **不完全 Dendron 化**(2026-07-01):dot 命名法对演示项目(HTML + 多文件)不适用,且缺工具支持
3. **采用文件夹 + MOC + 标签混合方案**(2026-07-01):扁平 + 二级分组 + MOC 策展,可扩展到 1000+ 篇
