### Python 代码
```python
# Python 快速排序
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

print(quicksort([3, 6, 8, 10, 1, 2, 1]))
```

## 4. 混合测试

下面是实际项目代码示例:

```tsx
// React Hook 示例
import { useState, useEffect } from 'react';

interface User {
  id: number;
  name: string;
}

export const UserList = () => {
  const [users, setUsers] = useState<User[]>([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch('/api/users')
      .then(res => res.json())
      .then(data => {
        setUsers(data);
        setLoading(false);
      });
  }, []);

  if (loading) return <div>加载中...</div>;

  return (
    <ul>
      {users.map(user => (
        <li key={user.id}>{user.name}</li>
      ))}
    </ul>
  );
};
```

对应的数据流图:

```mermaid
graph LR
    A[用户访问页面] --> B[组件挂载]
    B --> C[useEffect 触发]
    C --> D[发起 API 请求]
    D --> E{请求成功?}
    E -->|是| F[更新 state]
    E -->|否| G[显示错误]
    F --> H[重新渲染]
```
