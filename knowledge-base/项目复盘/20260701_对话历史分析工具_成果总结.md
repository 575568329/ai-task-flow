## 💡 核心发现

### 对话文件格式（JSONL）

每行一个 JSON 对象，包含：
```json
{
  "type": "user" | "assistant" | "tool_result",
  "message": {
    "role": "user" | "assistant",
    "content": "文本内容或数组"
  },
  "timestamp": "2026-07-01T01:33:24.905Z",
  "uuid": "唯一标识",
  "parentUuid": "父消息ID"
}
```

### 关键优化点

1. **过滤噪音**：排除工具调用结果，只提取真实对话
2. **处理大文件**：JSONL 逐行解析，支持大型会话
3. **双格式输出**：Markdown（可读性）+ JSON（可编程）
