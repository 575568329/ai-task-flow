
## 五、使用建议

### 📖 复习路径(推荐)

**第一轮(基础巩固,3-5天)**:
1. JavaScript 基础 → TypeScript → HTML/CSS
2. 边看边做笔记,手写高频题(闭包/原型链/this/Promise)

**第二轮(框架深入,3-5天)**:
3. Vue 3 / React(根据目标公司技术栈)
4. Node.js → Next.js
5. 浏览器原理 → 网络协议

**第三轮(工程化与优化,2-3天)**:
6. 构建工具 → 性能优化
7. 结合项目经验,准备"你做过哪些性能优化"

**第四轮(AI技术,可选)**:
8. LangChain → LangGraph → AI 大模型
9. 针对 AI 应用岗位

### 🎯 面试前一周冲刺

**每天复习 2-3 份文档**:
- 早上:看一遍文档(1小时)
- 中午:刷对应的高频面试题(30分钟)
- 晚上:手写核心代码(防抖/深拷贝/Promise/虚拟DOM 等,1小时)

**面试前一天**:
- 快速过一遍所有"高频面试题"部分
- 准备 3-5 个项目亮点(结合性能优化/难点攻克)

### 💡 进阶提升

**从八股文到实战**:
1. **跟着文档敲代码**:每个示例都自己敲一遍
2. **写博客**:把每个技术点用自己的话写一遍(费曼学习法)
3. **做项目**:用学到的技术优化现有项目(性能优化最容易体现)
4. **看源码**:Vue 3 响应式/React Fiber/Webpack 插件机制
